# Resources（资源库）默认文件夹模板

> 适用于：跨项目/跨领域的可复用资源
> 使用说明：存储通用模板、工具、学习资料

---

## 📁 默认文件夹结构

```
Resources（资源库）/
├── 01-系统工作流/        ← 全局通用工作流
├── 02-系统设计文档/      ← 系统架构
├── 03-工作记录/          ← Resources 层面记录
├── 04-模板库/            ← 可复用模板
│   ├── 项目模板/
│   ├── 领域模板/
│   └── 文章模板/
├── 05-工具与脚本/        ← 自动化工具
├── 06-学习笔记/          ← 书籍、课程笔记
│   ├── 书籍/
│   ├── 课程/
│   └── 文章/
├── 07-设计资产/          ← 图片、Logo
└── 08-代码片段库/        ← 常用代码
```

---

## 📋 各文件夹用途说明

### 01-系统工作流/
- 全局通用的工作流程
- 例如：
  - `整理对话记录工作流.md`
  - `记录工作工作流.md`
  - `工作日志管理工作流.md`
  - `结构变更同步工作流.md`
  - `qy-indicator-driven-project.md`

### 02-系统设计文档/
- PARA 系统架构说明
- Skill 设计文档
- 命名规范
- 使用手册

### 03-工作记录/
- **Resources工作记录_{年月}.md** - 记录系统层面的变更
- 配合 `/qy-record-work` Skill 使用

### 04-模板库/
- **项目模板/** - 项目文件夹模板
- **领域模板/** - 领域文件夹模板
- **文章模板/** - 常用文章模板（如：产品文案、教程模板）

### 05-工具与脚本/
- 自动化脚本
- 常用工具配置
- 快捷指令

### 06-学习笔记/
- **书籍/** - 读书笔记
- **课程/** - 在线课程笔记
- **文章/** - 精读文章的笔记

### 07-设计资产/
- Logo
- 配图
- 图标库
- 设计规范

### 08-代码片段库/
- 常用代码片段
- 配置文件模板
- API 调用示例

---

## ⚠️ 使用注意事项

1. **Resources 的定位**
   - 存储可复用的资源
   - 不属于特定项目或领域
   - 跨场景通用

2. **文件夹命名规范**
   - 使用数字前缀（01-08）保持顺序
   - 使用中文名称，易读易理解

3. **与 Skill 的配合**
   - `/qy-organize-chat` - 读取系统工作流
   - `/qy-record-work` - 读取系统工作流
   - `/qy-sync-structure` - 读取系统工作流
   - `/qy-analyze-workflow` - 分析系统工作流

4. **不要归档 Resources**
   - Resources 是长期维护的资源库
   - 不需要归档，持续更新即可

- **新建领域** - 从模板库复制领域模板
- **写文章** - 从模板库选择文章模板
